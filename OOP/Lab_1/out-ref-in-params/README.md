# Creating Static Methods

Beginner level task for practicing `out`, `ref`, `in` parameters, and tuples in the methods.

Estimated time to complete the task - 0.5h.

The task requires .NET 8 SDK installed.

## Task Description

1. Add `MethodsWithOutParameters` static methods to [MethodsWithOutParameters.cs](CreatingMethods/MethodsWithOutParameters.cs) file. See TODO comments there.
1. Add `MethodsWithRefParameters` static methods to [MethodsWithRefParameters.cs](CreatingMethods/MethodsWithRefParameters.cs) file. See TODO comments there.
1. Add `MethodsWithInParameters` static methods to [MethodsWithInParameters.cs](CreatingMethods/MethodsWithInParameters.cs) file. See TODO comments there.
1. Add `MethodsWithTuples` static methods to [MethodsWithTuples.cs](CreatingMethods/MethodsWithTuples.cs) file. See TODO comments there.
